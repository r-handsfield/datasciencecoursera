Question 2

Which of the following is a generic function in a fresh installation of R, with only the default packages loaded?

a) lm
b) dgamma
c) colSums
d) predict

Solution

Use /Library/Frameworks/R.framework/Resources/library/base/help/base-package, then search function names. Or, use /Library/Frameworks/R.framework/Resources/library/base/help/colSums (and others) to find package for each function.

Answer

Answer: colSums
